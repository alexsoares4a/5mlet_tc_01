Bem-vindo ao Tech Challenge - Fase 01

Este projeto faz parte do desafio proposto no curso de Machine Learning Engineering. O objetivo é criar uma API pública para consultar dados de vitivinicultura da Embrapa, abrangendo informações sobre produção, processamento, comercialização, importação e exportação. Esses dados serão usados futuramente para alimentar modelos de Machine Learning.
