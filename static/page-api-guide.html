Guia de Uso da API

1. Cadastro

Para acessar as rotas da API, é necessário criar uma conta e realizar a autenticação. Siga os passos abaixo:

  • Acesse a opção Cadastre-se no menu principal.
  • Preencha os campos com seu Nome de Usuário, E-mail, Senha e confirme sua senha.
  • Após o cadastro, um token de verificação será enviado para o e-mail informado.
  • Clique no link do token recebido para validar seu cadastro.

Importante: O cadastro só será concluído após a validação do e-mail.


2. Login

Para acessar as rotas da API, é necessário estar autenticado. Siga os passos abaixo:

  • Acesse a opção Login no menu principal.
  • Insira seu Nome de Usuário ou E-mail e sua Senha.
  • Após o login, você receberá um token JWT que será usado para acessar as rotas protegidas.

Dica: Salve o token JWT em um lugar seguro, pois ele será necessário para todas as requisições à API.


3. Testando as Rotas da API

A documentação interativa da API está disponível na guia Documentação da API (via Swagger). Além disso, você pode testar as rotas diretamente usando as URLs abaixo.

3.1. Produção

  • Descrição: Apresenta os dados de "Produção de vinhos, sucos e derivados".
  • Filtros disponíveis: Ano (entre 1970 e 2023).
  • Colunas retornadas: Produto, Quantidade (L).
  • Rota: /producao/{ano}
  • Exemplos de uso:
    • Dados de 1970: /producao/1970
    • Dados de 1980: /producao/1980
    • Dados de 2023: /producao/2023

3.2. Processamento

  • Descrição: Apresenta os dados de "Quantidade de uvas processadas no Rio Grande do Sul".
  • Filtros disponíveis: Tipo de uva e ano (entre 1970 e 2023).
  • Tipos de uva disponíveis:
    • subopt_01: Uvas Viníferas
    • subopt_02: Uvas Americanas e híbridas
    • subopt_03: Uvas de Mesa
    • subopt_04: Uvas Sem Classificação
  • Colunas retornadas: Cultivar, Quantidade (L).
  • Rota: /processamento/{tipo}/{ano}
  • Exemplos de uso:
    • Uvas Viníferas de 1970: /processamento/subopt_01/1970
    • Uvas Americanas e híbridas de 1999: /processamento/subopt_02/1999
    • Uvas de Mesa de 2023: /processamento/subopt_03/2023

3.3. Comercialização

  • Descrição: Apresenta os dados de "Comercialização de vinhos e derivados no Rio Grande do Sul".
  • Filtros disponíveis: Ano (entre 1970 e 2023).
  • Colunas retornadas: Produto, Quantidade (L).
  • Rota: /comercializacao/{ano}
  • Exemplos de uso:
    • Dados de 1970: /comercializacao/1970
    • Dados de 1985: /comercializacao/1985
    • Dados de 2023: /comercializacao/2023

3.4. Importação

  • Descrição: Apresenta os dados de "Importação de derivados de uva".
  • Filtros disponíveis: Tipo de derivado e ano (entre 1970 e 2023).
  • Tipos de derivado disponíveis:
    • subopt_01: Vinhos de mesa
    • subopt_02: Espumantes
    • subopt_03: Uvas frescas
    • subopt_04: Uvas passas
    • subopt_05: Suco de uva
  • Colunas retornadas: Países, Quantidade (Kg), Valor (US$).
  • Rota: /importacao/{tipo}/{ano}
  • Exemplos de uso:
    • Vinhos de mesa de 1970: /importacao/subopt_01/1970
    • Espumantes de 1999: /importacao/subopt_02/1999
    • Suco de uva de 2023: /importacao/subopt_05/2023

3.5. Exportação

  • Descrição: Apresenta os dados de "Exportação de derivados de uva".
  • Filtros disponíveis: Tipo de derivado e ano (entre 1970 e 2024).

  • Colunas retornadas: Países, Quantidade (Kg), Valor (US$).
  • Rota: /exportacao/{tipo}/{ano}
  • Exemplos de uso:
    • Vinhos de mesa de 1970: /exportacao/subopt_01/1970
    • Espumantes de 1999: /exportacao/subopt_02/1999
    • Suco de uva de 2023: /exportacao/subopt_04/2023

Dicas Gerais

  • Autenticação: Todas as rotas estão protegidas por autenticação.
  • Erros comuns:
    • Certifique-se de que o token JWT está válido e não expirou.
    • Verifique se o ano e os tipos de filtro estão dentro dos valores permitidos.
  • Documentação interativa: Para explorar todos os endpoints de forma visual, acesse a Documentação da API via Swagger.
